About

Soy Nicolás Butterfield, desarrollador Fullstack Junior con foco en construir aplicaciones web claras, mantenibles y pensadas para crecer. Trabajo principalmente con Angular, TypeScript y Django, combinando frontend moderno con backend sólido y bases de datos relacionales.

Tengo experiencia desarrollando sistemas en entornos institucionales, donde la estabilidad, la claridad del código y el mantenimiento a largo plazo importan tanto como que la aplicación funcione. Me interesa especialmente la arquitectura de aplicaciones, el diseño por features y las buenas prácticas que facilitan que otros desarrolladores puedan entender y reutilizar el proyecto.

Actualmente estoy construyendo este portfolio con Angular y Server-Side Rendering, no solo como presentación personal, sino como un proyecto open-source, pensado para ser explorado, forkeado y usado como base por otros desarrolladores que estén aprendiendo o buscando una estructura profesional.

Disfruto aprender tecnologías nuevas, mejorar procesos existentes y resolver problemas reales con soluciones simples y bien pensadas. Mi objetivo es seguir creciendo como desarrollador, aportando valor desde el código y la arquitectura.
